## 全文

We've added a new command to Claude Code called /insights

When you run it, Claude Code will read your message history from the past month. It'll summarize your projects, how you use Claude Code, and give suggestions on how to improve your workflow.

## 研究摘要

**TL;DR**：Claude Code 官方新增 `/insights` 指令，分析一個月歷史訊息，自動診斷工作流瓶頸並給出改進建議。

### 核心論點

- `/insights` 讀取過去一個月的 message history，不依賴外部追蹤工具
- 三項輸出：(1) 專案摘要（你在做什麼）；(2) 使用模式分析（你怎麼用 Claude Code）；(3) 工作流改進建議
- 功能定位從「執行工具」升級為「自我診斷顧問」，Claude Code 對自身使用方式有反思能力
- 由 Thariq（Anthropic Claude Code 核心成員）官方宣布，屬正式功能而非第三方插件

### 關鍵數據 / 工具

- 歷史讀取範圍：過去 1 個月的 message history
- 宣布者：Thariq（@trq212，Claude Code Anthropic 成員，211k followers）
- 觸發方式：`/insights` slash command
- 互動數：11592 likes，6948 bookmarks，1.2M+ views（高於平均，反映社群高度關注）

### 批判性觀察

- 推文未透露分析所耗的 token/成本，對重度使用者而言一個月歷史量可能相當可觀，初次執行前建議先確認 context 用量
- 「改進建議」品質高度依賴訊息歷史的完整度——若常用 `/clear` 清除上下文，歷史可能不連貫，建議品質未知
- 對 cc-workspace 用戶直接適用：可立即執行 `/insights` 驗證效果，與 `routines.md` 中的工作流自動化思路互補
