## 研究摘要

**TL;DR**：Anthropic 官方釋出 Claude Code Channels，透過 Telegram/Discord MCP 讓用戶從手機遠端控制 Claude Code session。

### 核心論點

- Claude Code 現在可透過 MCP 整合接受外部訊息控制，首批支援 Telegram 和 Discord
- 使用場景：長時間 agentic 任務跑在本機，用戶用手機透過通訊 app 監控與操控
- 官方釋出（Thariq 為 Claude Code 核心團隊成員），是產品功能正式公告
- 架構面：Claude Code 作為 MCP server 接收通訊 app 發來的指令，屬於 body 層擴展
- 影響：開發者無需離開 Telegram/Discord 就能啟動、查詢、指揮 Claude Code session

### 關鍵數據 / 工具

- 平台：Telegram、Discord（初始兩個整合）
- 推文瀏覽量：7,624,143（高度病毒傳播）
- 書籤數：17,936（代表大量用戶想嘗試）
- Thariq 追蹤者：211,363（Claude Code 核心成員，高可信度來源）

### 批判性觀察

- 功能本身對 cc-workspace 而言是已知資訊（`routines-remote.md` 中 Channels 段落已記錄），但此推文是官方首次公開宣告，具有時間節點價值
- 推文內容極簡（僅兩句話），技術細節為零——無設定方法、無 MCP 名稱、無 API 文件連結，需搭配官方文件補充才有實作參考價值
- 與 Remote Control（`/remote-control`）的定位差異未說明：Channels 比 Remote Control 更輕量（不需要 claude.ai/code），但依賴本機 terminal 開著
