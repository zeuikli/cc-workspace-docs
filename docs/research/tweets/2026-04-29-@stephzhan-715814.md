7/ Neural nets may become the host process.

Today, models run virtualized on classical computers.

But Karpathy imagines a future where the neural net does most of the heavy lifting...and CPUs become coprocessors for deterministic tool use.

Strange now. Maybe obvious later :) We'll be back!

## 研究摘要

**TL;DR**：Karpathy 的終極願景：神經網路成為主計算進程，CPU 退化為確定性工具呼叫的 coprocessor。

### 核心論點

- 當前架構：neural nets 在 classical computers 上虛擬化執行
- 未來願景：neural nets 主導大部分計算，CPU 成為確定性任務（工具呼叫）的 coprocessor
- 這是 Software 1.0/2.0/3.0 架構演進的終局想像

### 關鍵數據 / 工具

- 架構對比：今日（model virtualized on CPU）vs. 未來（neural net as host, CPU as coprocessor）

### 批判性觀察

- 這是思想實驗性的遠期願景，無近期可行動性；但對理解 agentic architecture 的方向有框架價值
- cc-workspace 的 Bash tool 正是 CPU coprocessor 角色的微型體現：LLM 決策，CPU 執行確定性操作
