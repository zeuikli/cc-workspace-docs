## 研究摘要

**TL;DR**：目前最完整的 Claude Code Skills 生態系地圖，1,116 個 skills 來自 500+ repo，分 13 個領域，附所有 GitHub 連結，是 Skills 探索與安裝的一站式索引。

### 核心論點

- Skills 是 Claude Code 最被低估的功能：大多數使用者不知道 Skills 存在，直接損失生產力
- 官方 + 合作夥伴 (Microsoft 133 skills、Sentry 30、Trail of Bits 29、Cloudflare 16) 已是企業級成熟度
- alirezarezvani/claude-skills 235 個是最大社群 collection，jeffallan 的 SRE/DevOps 66 個 skills 對本 workspace 特別相關
- 生態系仍在加速增長，目前採用具有複利優勢

### 關鍵數據 / 工具

- 1,116 unique skills，500+ repositories，200+ authors
- anthropics/skills：121,000 stars（17 official skills）
- Trail of Bits：4,700+ stars（security）
- Firecrawl：3,000+ stars（web scraping）
- Sentry：2,500+ stars
- jeffallan/claude-skills：66 個 SRE/DevOps/Terraform/K8s skills（與本 workspace 高度相關）

### 批判性觀察

- 文章格式為清單彙整，未提供安裝指令或相容性測試，部分 repo 可能已過時
- 1,116 這個數字是作者主觀計算，不同 skill 的品質差異極大
- Part 6 DevOps/Infrastructure 的 jeffallan/claude-skills 值得深入研究（SRE 背景高度相關）
