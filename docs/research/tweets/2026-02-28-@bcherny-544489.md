## 研究摘要

**TL;DR**：Claude Code 官方釋出 `/simplify`（PR 品質審查）與 `/batch`（平行代碼遷移）兩個新 Skill，由作者日常使用驗證。

### 核心論點

- `/simplify` Skill 負責 code review + 品質優化，自動化 PR shepherd 流程的一大核心工作
- `/batch` Skill 負責平行可拆解的代碼遷移任務（parallelizable migrations）
- 兩者合計涵蓋 PR 生命週期從撰寫到上線、跨模組批次遷移兩大工程痛點
- Boris Cherny（Claude Code 直接作者）個人日常使用，代表官方認可的使用模式
- 這是官方 Skill 而非社群貢獻，具有最高可信度與可攜性

### 關鍵數據 / 工具
