## 研究摘要

**TL;DR**：精選 10 個 Claude Code 生態系 GitHub repos，涵蓋官方工具、skills 庫、subagent 集合、記憶持久化與安全審查自動化。

### 核心論點

1. **官方資源值得直接採用**：anthropics/claude-code（55k+ stars）與 anthropics/skills（37.5k+ stars）是 skill authoring 的 canonical reference，多數 workspace 建設可直接對照。
2. **everything-claude-code 是工程化最完整的 harness**：30 agents、136 skills、60 slash commands、1282 tests、98% coverage，是衡量自建 workspace 規模的基準線。
3. **claude-code-system-prompts 提供版本追蹤**：155+ 版本 CHANGELOG，可用於逆向了解 Anthropic 內部 sub-agent prompts 演進，對 workspace 調校有參考價值。
4. **Repomix 解決大型 codebase 餵入問題**：支援 XML/Markdown/plain 輸出，適合將整個 repo 打包進 context，與 researcher sub-agent 搭配使用。
5. **claude-mem 補足跨 session 記憶缺口**：透過壓縮持久化 context，定位與 Auto Memory 互補（而非替代）。

### 關鍵數據 / 工具

- anthropics/claude-code：55k+ stars
- anthropics/skills：37.5k+ stars
- everything-claude-code：30 agents、136 skills、1282 tests、98% coverage
- Repomix：20.9k+ stars
- wshobson/agents：25k+ stars
- claude-squad：5.6k+ stars
- claude-code-security-review：2.8k+ stars
- 系列推文第 2 篇，backlog 尚有 27+ repos

### 批判性觀察

1. **列表型內容，缺乏實際評測**：所有 star 數與描述均為作者整理，未提供使用心得或 benchmark，無法判斷哪些 repos 實際成熟可用（例如 graphify 無 star 數，claude-mem 無說明壓縮策略）。
2. **everything-claude-code 數字驚人但存疑**：136 skills + 1282 tests 若屬實，是業界少見的工程化成果；但 hackathon 背景與長期維護品質通常有落差，引入前需評估 dependency 與更新頻率。
3. **對 cc-workspace 用戶的實際應用價值**：anthropics/skills 與 claude-code-system-prompts 最直接有參考價值——前者可比對 skill authoring pattern，後者可追蹤 sub-agent prompt 演進；其餘 repos 多為可選參考。
