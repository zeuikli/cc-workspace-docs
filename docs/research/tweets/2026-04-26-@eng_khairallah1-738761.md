## 全文

I tested 500+ prompts, and these are the 40 that produce expert-level output every single time.

定義：「A real prompt is an engineered instruction that produces consistent, high-quality, production-ready output regardless of which model you use. It includes the role, the context, the constraints, the format, the quality standard, and specific examples.」

跨模型測試：Claude、ChatGPT、Gemini 均驗證可用。

### Writing and Content (01 to 10)

**01. The Expert Article Writer**

```
You are a senior content strategist who has written for top-tier publications.

Write a [WORD COUNT]-word article about [TOPIC].

Audience: [WHO THEY ARE and WHAT THEY KNOW]
Angle: [YOUR UNIQUE TAKE]

Structure:
- Hook: Bold claim or surprising fact. No generic introductions.
- Problem: Why the current way of thinking is wrong or incomplete.
- Framework: 3-5 named sections with clear headers.
- Evidence: Each section must include one specific example or data point.
- Action: End with 3 specific things the reader can do THIS WEEK.

Rules:
- Paragraphs: 3 sentences maximum
- No filler phrases ("it is important to note", "in today's world")
- No hedge words ("might", "could potentially")
- Every claim must be specific
- Bold the most important sentence in each section

This article should be good enough to publish without editing.
```

**02. The Thread Architect** — Twitter/X thread 架構模板（含 Hook、Problem、Insight、Action 結構）

**03-10**: Content calendar builder, email sequence writer, SEO blog template, newsletter writer, LinkedIn post architect, case study creator, video script template, brand voice analyzer

### Analysis and Strategy (11 to 20)

**11. The Competitive Intelligence Brief** — 競品分析模板（含 positioning gaps、pricing、messaging）

**12. The Strategic Decision Framework** — 決策分析（列出選項、考慮因素、推薦）

**13-20**: Market entry analysis, customer persona builder, SWOT analysis, OKR creator, pricing strategy, product roadmap, go-to-market plan, business model canvas

### Technical and Development (21 to 28)

**21. The Code Review Expert**

```
You are a senior [LANGUAGE] developer reviewing production code.

Review this code for:
1. Security vulnerabilities (input validation, injection attacks, authentication)
2. Performance issues (database queries, memory usage, algorithmic complexity)
3. Maintainability (naming, comments, structure, single responsibility)
4. Edge cases not handled
5. Best practices violations

For each issue found:
- Severity: Critical / High / Medium / Low
- Issue: What is wrong
- Why: Why it matters
- Fix: Specific code correction

End with: The top 3 things to fix before this goes to production.
```

**22-28**: Architecture design, database schema review, API design, security audit, performance optimization, test writing, documentation generator

### Productivity and Personal (29 to 35)

## 研究摘要

**TL;DR**：40 個跨模型（Claude/ChatGPT/Gemini）通用 prompt 模板，涵蓋 6 個領域，最高價值是 Code Review Expert prompt（含嚴重程度分級和具體修復格式）和 Expert Article Writer 的反規則設計（no filler phrases、no hedge words）。

### 核心論點

- 「Real prompts」的定義：包含 role/context/constraints/format/quality standard/examples 的工程化指令
- Code Review Expert prompt 的價值：Critical/High/Medium/Low 分級 + 「top 3 to fix before production」是實用的工程 checklist
- 反規則設計（「No filler phrases」「No hedge words」「Paragraphs: 3 sentences maximum」）是提升 AI 輸出品質的精確控制手段

### 關鍵數據 / 工具

- 測試來源：500+ prompts 測試後精選 40 個
