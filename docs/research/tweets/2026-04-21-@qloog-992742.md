## 研究摘要

**TL;DR**：8 個高星標 Claude Code Skills 精選清單，按使用情境分類，涵蓋全能型、檔案處理、團隊角色模擬、Obsidian、行銷、科研等面向。

### 核心論點

- Skills 生態已有成熟的分類——不同職業/情境有對應的 Skills，而非一套適用所有人
- 星標數量級懸殊（superpowers 16萬 vs compound-engineering 1.5萬），頭部 Skills 壟斷效應明顯
- 名人背書效應：Garry Tan（YC CEO）、Steph Ango（Obsidian CEO）都有自己的 Skills——生態正向主流開發者蔓延
- 跨工具相容性（OpenClaw、Hermes）說明 Claude Skills 格式已開始成為標準

### 關鍵數據 / 工具

- superpowers（obra）：16萬星，讓 Claude 做事更嚴謹，brainstorm→spec→plan→TDD→review→merge
- anthropics/skills：12萬星，官方文件處理（Word/PPT/Excel/PDF）
- gstack（Garry Tan）：7.7萬星，模擬團隊角色進行審查
- obsidian-skills（Steph Ango）：2.5萬星，Obsidian 使用者專用
- claude-scientific-skills（K-Dense AI）：1.9萬星，科研/論文寫作

### 批判性觀察

- 星標數字明顯誇大（superpowers 原始 repo 約 16-18 萬星是可信的，但 gstack 7.7 萬星需要交叉驗證）
- 清單缺乏使用者驗證的質量評估，純以星標數作為排序指標
- 「适用于 OpenClaw 和 Hermes」的相容性聲明未提供任何設定說明
