## Top 10 事件

### 🔴 立即影響

**1. Claude 中文 Token 稅 1.71×（Wisely Chen AI 深度分析）**
- Claude 中文比英文多耗 71% token；OpenAI 1.15×；DeepSeek 0.95×（中文比英文便宜）
- 台灣金融客服 10K calls/天：Claude $2,310/月 vs GPT-4o $1,121/月 vs DeepSeek $102/月
- _SRE 行動_：**立即重算非英語 API 預算**；按敏感度分層選供應商

**2. AI Patch Wave 警報（NCSC + UK 官方）**
- Claude Mythos 已發現 2000+ zero-days（含 17 年 Linux RCE）；**99% 仍未修補**
- AI 發現速度 >> 人類修補速度 → 「已知未補」窗口急速擴大
- _SRE 行動_：訂閱 AI 輔助漏洞資訊源；加速 patch cadence；SLA 需更新

**3. GPT-Realtime-2 SOTA 語音 API（5/8 開放）**
- 128K context（↑ 4×）；Big Bench Audio 96.6%（+15pp）；Glean +42.9% 幫助率
- 並行 tool calls + adjustable reasoning（minimal→xhigh）；定價不變
- _關注_：語音代理成本架構需重新評估；call center 應用有實質 ROI 證明

**4. GPT-5.5 Instant 定價衝擊（$4/$12 per MTok）**
- 名義漲 60%；實際帳單 +49–92%（completion token 模式改變）
- Vision agents 比 structured APIs 貴 **45×**
- _SRE 行動_：審查所有以 GPT-5.4 為基準的預算預測；vision pipeline 改用 structured APIs

## 5/10–5/16 補充重大事件

### 🔴 立即影響

**11. Claude Code 定價政策逆轉（5/12–5/14）**
- 新政策：Claude 訂閱 credit 1:1 換算（$200 訂閱 = $200 API credit）；歷史 70–90% 折扣瞬間終止
- 第三方 harness（OpenClaw、T3 Code 等）用戶大規模取消訂閱；Theo 等知名開發者公開反對
- _SRE 行動_：**立即重新計算 Claude API 預算**；評估替代供應商（Gemini/GPT-5.5/DeepSeek）

**12. DeepSeek V4 Pro/Flash 正式開源（5/10）**
- V4 Pro：1.6T 參數（49B 激活）；V4 Flash：284B（13B 激活）；1M context；推理接近前沿模型
- 定價大幅低於西方供應商；預計 V4.1 6 月發布
- _SRE 行動_：評估分層部署（高敏感 → Claude/GPT；非敏感 → DeepSeek V4 Flash 大幅降本）

### 🟠 策略影響（1-3 個月）

**13. Codex Remote SSH + 行動應用（5/15–5/16）**

**14. Anthropic vs Pentagon + 蒸餾攻擊政策（5/10）**

- _關注_：AI 供應鏈地緣政治風險加劇；合規需求可能快速增加

**15. LangChain SmithDB + LangSmith Engine（5/12–5/14）**
- SmithDB：代理追蹤專用資料庫，查詢速度 12–15×；LangSmith Engine：trace 聚類 → failure ID → 自動修復建議
