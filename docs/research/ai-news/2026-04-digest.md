## 四大月度趨勢

1. **計算力集中** — OpenAI / Anthropic / Google 簽署數十億美元長期協議；邊界模型成本競爭加劇
2. **代理架構工程化** — Harness 成為獨立工程學科；持久、可驗證代理取代一次性演示
3. **控制層爭奪戰** — OpenAI 自製手機（2028）；Microsoft 關係重構；雲端 vs 裝置層重新調整
4. **安全「門檻化」** — Project Glasswing 限制性發布；代理沙箱成標配；供應鏈攻擊加速

Sources: the-rundown-ai 4/27, smol-ai-news 4/30
