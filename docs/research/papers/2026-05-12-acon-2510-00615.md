# ACON: Optimizing Context Compression for Long-horizon LLM Agents

## Abstract

The paper addresses a critical challenge in deploying language models as agents: managing expanding context lengths from accumulated action and observation histories. The authors propose ACON, a framework that "optimally compresses both environment observations and interaction histories into concise yet informative condensations."

The approach employs compression guideline optimization where capable LLMs analyze failures in paired trajectories to refine compression strategies. The authors additionally distill optimized compressors into smaller models to reduce computational overhead.

**Key Results:** The framework achieves "26-54% reduction in memory usage (peak tokens) while largely preserving task performance," maintains "over 95% of accuracy when distilled into smaller compressors," and improves smaller language models' performance by "up to 46%" on long-horizon tasks.

---

## Problem Statement

As LLM agents interact with environments across dozens of steps, accumulated action and observation histories become prohibitively large. This creates dual challenges:
- Elevated computational costs due to transformer scaling with context length
- Diluted attention from irrelevant historical information

---

## Core Contribution

ACON compresses both interaction histories and environment observations through optimized natural language guidelines rather than parameter fine-tuning. The approach employs "contrastive task feedback"—analyzing failures when compressed context is used versus successes with full context—to iteratively refine compression prompts.

---

## Method Overview

The framework operates in stages:

1. **Guideline Optimization**: Uses an LLM to analyze trajectory pairs and generate natural language feedback about what information was lost during compression
2. **Utility Maximization**: Refines guidelines to preserve task success
3. **Compression Maximization**: Further optimizes to reduce context while maintaining performance
4. **Distillation**: Transfers optimized compression capabilities to smaller models

---

## Key Technical Innovation

Rather than gradient-based optimization requiring parameter updates, the method uses gradient-free prompt optimization compatible with API-based closed-source models, making it deployable without model access.

---

## Experimental Results

Testing across three benchmarks (AppWorld, OfficeBench, 8-objective QA), ACON achieved:

- **Memory efficiency:** 26–54% reduction in peak tokens while largely maintaining task performance
- **Distillation effectiveness:** Over 95% performance preservation when distilled into smaller compressors
- **Performance improvement:** Improvements of 20-46% for smaller language models operating as agents

---

## Benefits

1. **Cost Reduction**: Significantly lower token consumption for long-horizon tasks
2. **Scalability**: Compatible with API-based closed-source models without requiring parameter updates
3. **Transferability**: Compression strategies distill effectively to smaller models
4. **Generalization**: Optimized guidelines preserve core task information while removing noise

## Implications

The work demonstrates that intelligent compression of accumulated agent histories can maintain high performance while dramatically reducing computational overhead. This is particularly valuable for long-horizon multi-step tasks where context accumulation is a primary bottleneck in both cost and attention quality.
