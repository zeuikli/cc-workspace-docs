# A-MEM: Agentic Memory for LLM Agents

**ArXiv**: 2502.12110 | **Date**: 2025-02-17 (修訂 2025-10-08) | **Venue**: NeurIPS 2025 | **Authors**: Wujiang Xu, Zujie Liang, Kai Mei, et al. | **Category**: cs.CL

## 摘要

解決 LLM agent 記憶系統的關鍵差距：「當前記憶系統能夠基本存取和檢索，但缺乏複雜的記憶組織。」採用 Zettelkasten 筆記方法論（強調互聯知識）結合 agent 驅動決策，實現動態記憶組織。

## 核心方法

### 動態記憶組織

不同於靜態結構，新資訊進入時：
1. 生成包含多結構屬性的完整筆記（上下文描述、關鍵詞、標籤）
2. 分析歷史記憶識別相關連接
3. 在相關概念間建立有意義的連結

### 記憶演化機制

新記憶整合時可觸發**現有記憶表示的更新**，使系統對知識庫的理解持續精化。

## 主要結果

在使用 6 個基礎模型的評估中，超越現有 SOTA 基準。

## Zettelkasten 方法論

德語「卡片盒」，一種知識管理系統，核心原則：
- 原子化筆記（一個概念一個筆記）
- 互聯引用（概念間顯式連結）
- 湧現結構（從連結中湧現的高層理解）

將此原則應用於 agent 記憶，使記憶不再是孤立向量點，而是互聯的知識網絡。

## 與其他記憶系統的對比

| 系統 | 組織方式 | 演化機制 |
|------|---------|---------|
| 純向量 RAG | 語義相似性 | 無 |
| HippoRAG | KG + PageRank | 部分 |
| **A-MEM** | Zettelkasten 互聯 | **動態更新現有記憶** |

## 分類

**Memory Architecture** — NeurIPS 2025 論文，Zettelkasten 式互聯記憶組織是向量 RAG 之後的重要進化方向，動態記憶演化機制值得在生產環境記憶系統中考慮。
