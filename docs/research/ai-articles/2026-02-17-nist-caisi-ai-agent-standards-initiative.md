# AI Agent Standards Initiative

> **來源**：[nist.gov](https://www.nist.gov/caisi/ai-agent-standards-initiative)
> **作者**：NIST CAISI（Center for AI Standards and Innovation）
> **發布日期**：2026-02-17
> **收錄日期**：2026-05-27

---

## Overview / 概要

NIST 的 AI 標準與創新中心（CAISI）於 2026 年 2 月 17 日正式啟動 AI Agent Standards Initiative，這是美國聯邦政府首次針對自主 AI agent 建立互通性、安全性與身份標準的系統性努力。目標是確保下一代 AI（具備自主行動能力的 agent）能被廣泛、有信心地採用。

---

## Three Strategic Pillars / 三大戰略支柱

### 1. Industry-Led Standards（產業主導標準）

- NIST 主辦技術會議、進行標準缺口分析，產出自願性指引
- 與跨機構夥伴（包括 NSF）合作，推動美國在 ISO/IEC JTC 1 等國際標準機構的領導地位

### 2. Community-Led Protocols（社群主導協定）

- 與利害關係人合作，降低 agent 互通協定的採用障礙
- NSF 投資開源 AI agent 生態系統的建設與維護

### 3. Research Investment（研究投資）

- NIST 開展 agent 認證、身份基礎設施的基礎研究
- 為協定開發建立安全評估方法論

---

## Key Deliverable: Agent Interoperability Profile（Q4 2026 計畫）

**AI Agent Interoperability Profile** 是本計畫首個完整規範性輸出，預計於 2026 Q4 發布。該 Profile 將涵蓋：

- Agent 身份與授權的標準化框架
- 跨系統 agent 互通的技術規範
- 安全與風險管理的自願性指引
- 監控與日誌記錄要求

## Scope & Context / 範疇與背景

現代 AI agent 能力已涵蓋：

- 自主撰寫與除錯程式碼
- 管理通訊與行事曆
- 執行商業交易

這些能力使 agent 從工具轉變為自主行動實體，現有的安全與身份框架並非為此設計。本計畫旨在填補此框架缺口。

---

## Collaborating Partners / 合作夥伴

- **NSF**（National Science Foundation）：共同投資開源協定生態
- **ITL**（Information Technology Laboratory, NIST）：AI Agent 身份與授權概念文件
- **NCCoE**（National Cybersecurity Center of Excellence）：開發將身份標準應用於企業 agent 實作的指引
- 各聯邦機構（通過跨機構協調機制）

---

## Significance / 重要性

本計畫代表：

1. **首個聯邦層級** 針對自主 AI agent 的標準化努力
2. **三軌並行**：標準（NIST/ISO）+ 協定（開源社群）+ 研究（基礎安全）
3. **自願性框架**為先，為未來法規提供技術基礎
4. **國際領導戰略**：透過 ISO/IEC 確保美國標準在全球 AI agent 互通規範中的主導地位
