## 核心摘要（繁體中文）

該儲存庫是關於 Claude Code 使用的全面教育資源，強調深度理解而非單純配置複製。包含 24,000+ 行參考文檔、48 個 Mermaid 圖表、271 道測驗題目和 181 個生產範本，特別強調安全性和威脅情報。

### 五項黃金法則

1. **驗證信任**：Claude Code 產生的邏輯錯誤率較高，需要多層驗證
2. **審查 MCP**：來自未知來源的伺服器存在安全風險
3. **管理上下文**：70% 使用率時精度開始下降，性能衰退明顯
4. **漸進採用**：從簡單開始，逐步擴展功能和複雜度
5. **使用方法論**：TDD/SDD/BDD 對 AI 協作至關重要

### 開發方法論

**Test-Driven Development（TDD）**
- 先寫測試，再實現代碼
- AI 代理生成的代碼品質更高

**Specification-Driven Development（SDD）**
- 詳細規格優於含糊需求
- 規格應包含邊界條件和異常

**Behavior-Driven Development（BDD）**
- 用業務語言定義行為
- 便於 AI 理解上下文

**Get Shit Done（GSD）方法**
- 實用的務實方法
- 在時間緊張時選擇此方法
