# 如果你想用一年學好寫作，這些事你一定要知道

**來源：** https://stevekoh.substack.com/p/b65

---

如果你在一年前跟我說：「你會靠打字過活。」

我一定不會相信。怎麼可能？靠打字就能過生活？

但現在，我就在這裡。寫了超過 50 篇文章，靠線上寫作賺錢，還開始教人怎麼做一樣的事。

這不是我當初設想的路，卻是我現在最踏實的生活方式。

人生就是這樣。你永遠猜不到它要怎麼展開。

我從來沒覺得自己寫得特別好，也不是那種充滿靈感的創作者。

我大學念的是 Business Information System，一個講起來聽起來好像很有前途的科系。但到最後一年，我幾乎撐不下去，甚至差點退學。

那時候我已經很確定這不是我要的路。

寫作一開始只是我在線上賺錢這條路上，無數嘗試中的其中一項。

賣產品、自創品牌、做顧問、廣告服務，線上賺錢的方式幾乎我都試過。

但寫作是我唯一會一直想回來做的事。

因為它自由，誠實，讓我每天都能和世界對話。

寫字不需要辦公室，不需要團隊，不需要開會。

只需要你和你的腦袋，還有那個願意打開來聽你說話的陌生人。

這種感覺很純粹。

現在52週過去了。我從這一年裡學到了什麼？值得嗎？

值得，非常值得。

如果要我回到起點，重新開始，我會給當時的自己5條關於天賦、關於心態、關於建立受眾群與關於關於練習寫作賺錢的提醒。

# 1. 關於天賦

我們從小被教育要發揮優點、改正缺點。

但你可能從未想過：優點和缺點，其實都是天賦的不同表現方式。

我從來沒有打算讓所有人都喜歡我。

不要讓討厭你的人，否定了你的天賦。

也不要讓你喜歡的人，限制了你的可能。

所以，你不是沒有優點，也不是沒有天賦。

你只是還沒認真整理過它們，或者還不懂怎麼用對的方式發揮它們。

你可以今天就拿出一張紙，安靜地回答這些問題幫助你找到自己的天賦：
- 你現在做的事，是你擅長也熱愛的嗎？如果不是，你認為你擅長什麼？

- 你真正喜歡的興趣是什麼？（不是為了錢，不是為了證書）

- 別人常常找你幫忙的是哪類問題？

- 如果你現在財富自由，明天就不用上班，你會做什麼？

寫完之後，你會知道自己怎麼看事情，喜歡怎麼活、適合往哪裡走。

# 2. 關於心態

很多人以為開始寫作的最大難關是內容，但真正的挑戰是你必須學會管理自己與世界之間的關係。

首先是你與平台的關係。

寫作是創作，發表是發行。

今天的發行平台是什麼？

是 Instagram、是 Threads、是 YouTube、是你選的那個能讓你被看見的地方。而平台背後的機制叫演算法。

所以第一個必須建立的心態是：你不能依賴演算法，但你也不能敵視它。

每一次平台改動，許多創作者像地震來了般焦慮：觸及下滑了、內容沒人看了、是不是要轉戰新平台？

但事實是平台從來不是為了成就你而設計的，它的使命不是幫你紅起來，而是讓用戶留下來。

你越早接受這個現實，就越能冷靜作戰。

平台不是你的敵人，也不是你的救世主，它是一個合作對象。

與其抱怨變更，不如去理解它，掌握它，然後為我所用。

若有一天不再適合你，那就瀟灑地離開，而不是戀戰。

第二個心態挑戰更艱難：你無法避免被討厭。

寫作的本質，是把腦中最私密的東西公開，供人閱讀、評價、甚至批判。

這意味著無論你多真誠，還是會有人曲解你、嘲諷你、否定你。

這不是你做錯了，而是這個世界的結構如此。

你讀者越多，得到的惡意也越多。

你想當一個被很多人喜歡的人，那你也得承受被少數人強烈厭惡的代價。

酸民是一種稅，你在觀眾中賺得的注意力，得繳一部分給他們。

這兩個心態，一個關乎穩定，一個關乎堅持。

穩定是你在平台浮動的環境下，依然能維持輸出節奏。

堅持是你在面對惡意與冷場時，依然選擇繼續寫。

只有你挺過去，作品才會留下來。

# 3. 關於練習

你可能會說：「他是因為幸運，才有那麼多讀者。」

但如果你真的深入觀察，那些你稱為幸運的寫作者，其實早在你看見他之前，就已經默默累積了無數次的練習和試錯。

寫作不是靠靈感，不是靠運氣，靠的是一次次坐下來，把腦中的混亂提煉成文字。

很多人以為成功的關鍵是自律，但比自律更重要的，是你跌倒後還願不願意再次站起來，哪怕只是寫下一句話、發出一段文字，也算是一種回歸。

寫作的過程不是一條直線，而是無數個「寫不好 → 優化 → 再寫一次」的循環。

每一次迭代都是一次進步的可能。

你會慢慢發現自己的觀點更有力量，文字更有穿透力，讀者也因此開始留下來，開始回應你。

別擔心你現在的寫作還不夠好，真正該擔心的是你是不是還在原地觀望，還沒開始動筆。

# 4. 關於建立受眾群

幾十年前，「靠出書過生活」這條路的確是寫作者唯一的選擇。

你得仰賴出版社認可、編輯青睞，才有機會讓你的作品被看見。

但今天的世界，早已不同。

現在，你可以自行出版電子書、經營社群媒體，甚至直接提供服務賺取收入。

在還沒獲得主流認可之前，你就能自己開始說話、累積聲量、打造影響力。

你不再需要等別人開門，因為網路時代，你可以自己蓋一棟房子，吸引讀者自己走進來。

現在的你，與過去的「寫書為生」的作家有何不同？

你是一個超級個體。

你有能力建立個人品牌、拓展多元收入來源，而不再只是等待某天某人給你一個機會。

如果你願意開始，願意反覆學習，那麼你就可以擁有一群真正願意為你作品買單、為你理念共鳴的讀者。

# 5. 關於寫作賺錢

很多人說：「我想做自己喜歡的事，然後靠這個賺錢。」

這句話本身沒錯，但順序要調整過來才會運作。

不是「我喜歡 → 所以賺錢」，而是「這東西有價值 → 然後我也喜歡 → 所以可以持續 → 所以能賺錢」。

如果你只靠喜歡在寫作，遲早會遇到兩種挫折：
- 內容雖然你很愛寫，但沒人願意為它買單

內容雖然你很愛寫，但沒人願意為它買單
- 沒錢撐住你的創作，你終究會放棄它

沒錢撐住你的創作，你終究會放棄它

寫作若要變成事業，你就得學會問這三個問題：

i. 我的內容能解決誰的什麼問題？

這不是要你迎合大眾，而是你得釐清你的寫作是否能產生實際價值。

娛樂是價值、療癒是價值、教育是價值。

但都需要被明確定義，否則你只是在日記。

ii. 我可以怎麼建立與讀者的持續關係？

不要只追求一次性觸及。

真正的商業價值來自關係。

你是否擁有一群願意持續聽你說話的人？

電子報名單是最低成本、最高信任的方式之一。

不要等到你很有影響力才開始收名單，那就像說你想等自己瘦下來才去健身。

iii. 我的作品如何轉化為產品或服務？

你無法靠純內容賺錢，除非你是報社或媒體公司。

變現方式往往來自內容之外的延伸：教學、顧問、數位產品、社群。

寫作不會讓你賺錢，寫作帶來的影響力才會。

你不能只是每天寫一篇貼文然後期待有人捧著錢來說「我想付費給你」。這不會發生。

你要讓對方知道：「這裡有免費內容讓你了解我，這裡有付費內容讓你成長。」

這是寫作者最該學的銷售邏輯：先貢獻，再交換。
