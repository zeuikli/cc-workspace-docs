# 普通人如何把握好25～35歲這關鍵十年

**來源：** https://stevekoh.substack.com/p/1-ddb

---

1. 重啟人生最快的方式：早起、呼吸、思考、閱讀、寫作、運動、冥想、學習、日記、戒癮、創作、反省、感恩、計劃、正念、放鬆、適應、利他、靜心、極簡、熱愛。

2. 銷聲匿跡1年 + 定義願景 + 系統化思維 + 持續創作 + 掌控時間（8小時睡眠 + 4小時深度工作 + 1小時運動 + 30分鐘閱讀 + 15分鐘冥想 + 10分鐘日記）= 成為超級個體

3. 找個安靜的地方狠狠藏。每天拿出1小時瘋狂學習演講、寫作、銷售、創作，持續閱讀，重構自己。學會拒絕，學會爭取，學會無視他人的無謂評價。6到12個月後，你會發現，95%的人早已停下來了。

4. 把問題寫下來，正視你的焦慮源頭，才有機會拆解它。逃避情緒，只會讓它越來越強大。一旦進入心流，你會找到熱情，駕馭焦慮，征服挑戰。

5. 慢慢來、不著急。不過度投入、不綁死結果。太想得到的，反而失去得快。太用力的關係，容易走樣、走偏、走散。做對的事，不需衝刺，靠的是耐力與心力。生活中，10%是事件，90%是你的反應。活得好的關鍵，從來不是外在的事情。

# 6. 整明白人生的優先級

很多人折騰一生，都沒整明白人生的優先級

找到問題本質，像高手一樣無痛自律。

22個問題，弄清楚想過怎樣的人生：

1. 我是誰？我為什麼來到這裡？

2. 我希望在離開後被人如何記住？ 我想為之做出什麼貢獻？

3. 如果我實現了所有人生目標，我會有什麼感覺？

4. 別人欣賞我什麼？ 我想與世界分享哪些獨特的才能？

5. 對我來說什麼最自然？ 我的天才領域是什麼？

6. 我的五大價值觀是什麼？

7. 您有哪些鮮明的個性特質？

8. 您的穿著/說話/行事如何？ 您如何展示和表現自己？

9. 您如何描述您的氛圍？ 當你走进一个房间时，其他人会从你身上感受到什么样的能量？

10. 人們通常如何談論你？ 您認識的人如何描述您？

11. 如果您正在顯化名聲或公众認可，您的粉絲如何看待和描述您？ 您如何激励他们？

12. 您的日常習慣是什麼？

13. 您每天感觉如何？

14. 您擅长什么？ 您的独特优势和才能是什么？

15. 您是否相信自己？ 您如何思考和感受自己以及您的能力？

16. 為什麼成功對您來說如此容易實現？

17. 當你在朝著某個目標努力時，如何看待出現的障礙和挑戰？

18. 你是否將它們視為阻礙還是提升自己的機會？

19. 你的早晨活動讓你在心理和生理上有何感受？

20. 你是一個好人嗎？你有善良的意圖嗎？

21. 你是如何幫助別人的？

22. 你最感激的是什麼？

# 7. 一個人的很大的能力就是慢 + 靜

一個人的很大的能力就是慢 + 靜

走路慢一點，

讓自己靜得下來，

每一步走路聲都頓時變得清晰。

呼吸也開始通暢起來，

像久違的深呼吸，

突然好像沒這麼焦慮了。

原來靜得下來才是真正的奢侈品。

# 8. 人生，如果找不到方向，那就走向自己

人生，如果找不到方向，那就走向自己

藏起來好好：

療癒創傷

調節神經系統

戒掉所有成癮

重新養育你的內在小孩

走得慢也好，步子小也好，往前走就好。

# 9. 別用別人的時間表衡量自己

容易焦慮的人都有一個不太健康的習慣

幻想自己能控制一切。

焦慮不會讓事情變快，行動才會。

你越想趕路，反而越走不穩。

別用別人的時間表衡量自己。

有人25歲爆紅，有人45歲才翻盤。

你的時區裡，沒有遲到。

明白什麼道理後你不再焦慮了：

我以為個人成長就是不斷追求成功，直到我在衝刺的路上撞上了心理障礙，才發現心態崩了，再多技巧都沒用。

很多道理，我們讀的時候點頭如搗蒜，真的遇上時卻瞬間忘光。

因為明白不是從書裡來的，是從日子裡摔出來的。

# 10. 人的命運啊來自於念頭

人的命運啊來自於念頭

一念起萬物生，一念落萬物滅，

從起心動念的那一刻起，結局便已確定。

每天只聚焦三件事情，永遠不要超過三件：

1. 下指令：念頭決定你的意識走向

剛醒來時，大腦還處於 α 波與 θ 波之間，

這是潛意識最敞開的黃金時段。

此刻你接觸的第一個訊息，

會在你體內種下一整天的信念。

睜開眼後，閉目 30 秒，對自己說出關鍵句子：

• 我今天選擇專注與前進

• 我只做真正重要的事

避免一醒來就滑手機，因為那是在把注意力交給別人。

2. 做深度任務：打造聚焦迴路

如果你在早晨立刻進入閱讀、運動或創作，

你的大腦會進入一種清晰又穩定的聚焦狀態。

這種聚焦會自動觸發一個正向增強回路：

行動 → 清晰 → 成就 → 動力 → 更多行動

你可以安排一項零干擾的深度任務（寫作、專案、閱讀）

無需長，1小時足矣，關鍵在於質量與獨處感。

3. 結束前反問：今天我為什麼而活？

我們不是靠「做很多事」變強，

而是靠「做對的事」形成內在聚焦。

早晨結束前問自己一個問題，

你會開始為自己的人生活：

寫下今天一個關鍵目標：「我今天最想完成的是＿＿＿」

回到初衷，而不是盲目追趕待辦事項。

給它一個方向（下指令）

給它一個任務（深度專注）

給它一個理由（意義聚焦）

當你掌控了早晨，你就掌控了整天的能量走向。

不是因為你意志力強，而是你用了對的方法。
