---

你早起過，計劃過，執行過，也突破過。

但不知道為什麼，你還是卡住。

那種感覺一直不對。

你不缺努力的方法。你聽過足夠多的道理，制定過足夠多的計劃。

你缺的，是那個讓一切開始流動的底層開關。

那個開關，不在你的時間管理系統裡，不在你的任何一個執行框架裡。

真正讓你卡住的，是你還在用一個舊版的自己的邏輯在做選擇。

那個版本的你，是在別人的設定裡長大的。

父母的期待，學校的評分，社會的眼光。

你帶著那一整套應該和必須走進了你的人生。

然後你以為，那些是你的。

它們從來都不是你的。

我要給你8個很玄學的成事秘訣。

之所以說玄學，是因為這些話和你被輸入的那套系統完全相反。

它們不會告訴你怎麼更努力。

它們會告訴你，為什麼你一直這麼累，以及你可以怎麼停下來。

現在，我們先進入第一個秘訣。

# 1. 真心建議：不要看與自己無關的訊息

去愛自己，把時間和精力都專注在自己身上。

注意力力是宇宙唯一的貨幣。當你只專注自己，奇蹟就會開始發生。

把自己當成什麼人，就會慢慢變成什麼人。

不要看和自己無關的訊息。現在你看到的 99% 的訊息，都是：• 刻意吸引你眼睛的刺激• 用負面情緒抓住你注意力的餌• 想讓你分心的噪音它們沒有要讓你變成更好的自己，只想讓你被餵養，就慢慢失去選擇力。你的人生本來是主角視角，但當你一直活在旁觀者的人生、觀眾的心態，慢慢變成逐漸失去自己。能量來自於選擇，選擇來自於邊界，邊界來自於拒絕。拒絕那些與自己無關的訊息，就是在保護你的注意力、保護你的直覺、保護你的生命力。最後終於不再把生命浪費在不屬於你的人身上。專注力低的人，請好好愛自己。不要看與自己無關的信息，把時間和精力都專注在自己身上。算法時代，真正的自由是注意力自由。放下手機，保護你寶貴的注意力。

銷聲匿跡1年 + 定義願景 + 系統化思維 + 持續創作 + 掌控時間（8小時睡眠 + 4小時深度工作 + 1小時運動 + 30分鐘閱讀 + 15分鐘冥想 + 10分鐘日記）= 成為超級個體你的注意力在哪裡，人生就在哪裡生長。注意力是一種貨幣。它流向哪裡，哪裡就會被放大。當你不停地反芻，就等於在投資負能量。你在給過去充值，卻期待未來升值。能量管理的最高境界，是選擇性遺忘。把注意力收回來，用在輸出、學習、創作、鍛鍊、沉澱、突破。你越專注於建構當下，過去的陰影就越無力。

# 2. 你会擁有你想要的一切

用量子力學來解釋一下吸引力法則，你現在之所以想要，是因為未來已經擁有。

你之所以如此渴望某樣東西，是因為它在時間線上的某個時刻已經是你的了，是未來的你在提示現在的你，你值得擁有一切美好。擔心是最沒用的提前準備。你現在擔心的那些事，有八成，不會發生。而那剩下真正會發生的，通常不是你擔心的那個。是你完全沒想到的那個。所以你花在擔心上的每一分鐘，既沒有阻止任何事，也沒有讓你更準備好。只是把你搞得很疲憊。把自己養好。身體好。心情好。判斷力清晰。這才是最實際的準備。其他都是消耗。你接納什麼，什麼就消失，你對抗什麼，什麼就存在。你真正的敵人是你自己。你所經歷的一切，全部都是你內心世界的映照。心一釋然，萬物皆安。你允許讓風是風，讓雨是雨。讓別人按照他們自己的方式活著。讓事情按照它本來的樣子發生。人生本來就是不斷釋懷，不斷豐盈。順境來了，感恩。逆境來了，歷練。歡喜留下，遺憾也留下。從容面對每一段，才是真正活過。你根本不可能錯過任何註定屬於你的事物。屬於你的，永遠不會錯過。因為它本來就是為你而來的。時機到了，它自然出現。所以你不可能失去任何原本就屬於你的東西。那些離開的，本來就不是。那些還沒來的，還在路上。那些你以為錯過的，也許只是還沒到你準備好的那一刻。放下焦慮。因為一切都在被安排著好起來。

# 3. 《金剛經》最孤獨的一句話，可以讓你瞬間開悟

應無所住而生其心。

不執著於某一個樣子，所以什麼樣子都能是你。

開放。融合。接納。允許。

你本自具足，不需要任何人的認同來讓你完整。不需要在任何幻影裡尋找寄託。失去認可的那一刻，你還在。被誤解的那一刻，你還在。被否定的那一刻，你還是你。你的自性，從來沒有動搖過。動搖的，只是你以為自己需要那些東西的錯覺。所有事情的發生，那些讓你痛的，讓你困惑的，本質都是讓你明心見性的考驗。每一件事是鏡子，都在照給你看。你還抓著什麼，你還怕失去什麼。當你不再執著於固化某個狀態，不再堅持我就是這樣，不再抓著事情應該是那樣，每一個當下，都可以是全新的誕生。每一次相遇，都是你修行的助緣。每一次失去，都是你鬆開手的練習。當你能夠跳出肉身，站在更高的位置往下看。那些讓你輾轉難眠的紛爭，那些反覆在腦中播放的評論，那些別人對你的定義和標籤，你會看見它們有多小。

# 4. 你越感恩就越是強大

睡前，感謝三件事。可以很小的事。今天的咖啡很好喝。有人對你笑了一下。你的身體撐過了今天。你的大腦就開始學會在每一天裡，主動去搜尋那些美好的時刻。你開始先看見有什麼，而不是先看見缺什麼。這一個轉換，會改變你感受整個生命的方式。一個人越能看見自己擁有什麼，好運就越知道，該往哪裡靠近。一個緊握著匱乏感的人，即使好事發生，他也感受不到。一個習慣感恩的人，即使只是一杯好喝的水，他都能從裡面接收到生命的善意。你感恩什麼，那些就會慢慢變多。你的注意力放在哪裡，你的能量就流向哪裡，你的生命就長向哪裡。越是感恩的人，越容易遇見好事。你帶著感謝活著，你發出的能量就不一樣了。你吸引來的，也就不一樣了。宇宙會不斷給你更多理由去感恩。感恩，是最溫柔的吸引力法則。只需要你真心說一句謝謝。謝謝這一天。謝謝這個當下。謝謝那些你以為理所當然、卻其實都是禮物的事。先感恩，一切才會越來越好。一個人越常說謝謝，他的生命就越活得越順。因為他相信。認真。堅持。感恩。相信的人，抓住了別人看不見的機會。認真的人，在每一個細節裡改變了自己。堅持的人，用時間重寫了命運的劇本。感恩的人，留住了福報。因為他知道自己擁有什麼，所以那些好的，捨不得離開他。

# 5. 你越放鬆，結果就會越好

人只有在快樂的時候，運氣才會變好。你越快樂，自然吸引高頻率的事物靠近。修心是讓自己越來越安靜。是讓自己越來越平和。是不管外面發生什麼，你內心還是平靜的。當你的磁場在往上走，能量是正向流動的，你不需要追，不需要搶，不需要拚命去抓，美好的事情自己會來。保持快樂，是你選擇用最高的頻率，活在這個世界上。為什麼人一旦過得輕鬆，反而開始心虛？因為我們太早學會了一種錯誤的等式：疲憊才配得上價值，忙亂才稱得上努力，安靜與從容，彷彿自帶原罪。於是你一閒下來，內心就開始自動審判。好像沒有焦慮，就等於沒有在前進，沒有壓力，就等於正在浪費人生。但放鬆不是墮落。那是身體在修復，是能量在回流。當你心情好的時候，腦子轉得特別快，靈感也來得特別自然。正向情緒會直接打開你的認知彈性。快樂能抵銷壓力對神經系統的侵蝕。當你放鬆下來，運氣才有縫隙可以進來。焦慮是因為你想同時做太多事，又急著看見結果。這種拼命向前衝的能量，是在消耗你、堵住你。修心是讓自己更平靜。是允許能量自然流動，而不是用意志力強行推它。當你不再抓，你要的東西反而開始靠近。最頂級的爽感就是完全專注於自己。這樣的人，是不可能內耗的。因為他幾乎不鳥任何人。他不需要任何人的認可，不需要跟誰交代，更不會讓別人的負面評價在他心裡住超過三秒。外界的噪音是別人的事。當你不需要，你就自由了。不需要被肯定，就沒有失落。不需要被喜歡，就沒有討好。不需要任何人給你一個說法，就沒有煩惱。那種自給自足的狀態，才是真正的爽。能把自己照顧好，本身就是一種穩定而乾淨的高能量。吃飽。喝足。睡夠。就是這麼基本。無論你現在面對什麼，先養好你的精氣神。在你有力氣之前，任何解法都只是空談。吃飽，才有力氣做事。睡夠，才有清醒去思考。那些你以為過不去的事，在你睡一覺醒來、吃一頓好飯之後，往往會發現沒那麼大不了。
