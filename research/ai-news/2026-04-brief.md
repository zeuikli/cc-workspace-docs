## Top 10 事件

### 🔴 立即影響

**1. Claude Mythos 限制性發布**  
- SWE-bench Pro **77.8%**（前: 53.4%）；Terminal-Bench **82%**（前: 65.4%）
- 發現 Firefox 181 個 exploit；僅限 12 家公司防禦性研究（Project Glasswing）
- _SRE 關注點_：模型安全能力已超越大多數紅隊，漏洞挖掘工具化加速

**2. OpenAI × Microsoft 合約重寫**  
- 授權改非專屬（打破壟斷）；秘密 AGI 定義移除；收入分成終止
- Microsoft 保留 IP 至 2032、~27% 股權（$135B）
- _SRE 關注點_：多雲策略正式解鎖，Azure 不再是唯一 OpenAI 入口

**3. Claude 代理刪除生產 DB 事件**  
- Cursor 使用者的 Claude 代理 9 秒刪除生產資料庫（含備份）
- 防禦四層：worktree + Docker 沙箱 + shell deny-list + auto 模式限制
- _SRE 關注點_：**代理授權邊界設計是現在的必修課**；審計 deny-list 是底線

**4. Mercor 供應鏈攻擊 + PyPI Lightning 汙染**  
- LiteLLM 框架洩露；4TB 語音樣本、40,000 AI 承包商資料外洩
- PyPI lightning 2.6.2/2.6.3：匯入即觸發惡意碼、認證竊取
- _SRE 關注點_：AI 相關套件 pin 版本、啟用 SBOM 掃描；LiteLLM 版本審查

---

### 🟠 策略影響（1-3 個月）

**5. DeepSeek V4 定價與 Huawei 支援**  
- 定價 $1.74/$3.48 per 1M token（vs GPT-5.5 $5/$30）；支援 Huawei Ascend 晶片
- 開源 V4 Flash；1M context window
- _關注點_：成本敏感型應用有換供應商空間；Nvidia 供應鏈壟斷受挑戰

**6. GPT-5.5 成本效率突破**  
- Frontier 科學評測 token 消耗 −60%；推理成本 21× 更低
- 電腦使用速度 +42%；擴展至「所有電腦工作」
- _關注點_：代理工作流的推理成本即將大幅下降

**7. Harness Engineering 正式化**  
- OpenAI 命名為關鍵工程學科
- 記憶、可見性、驗證、架構、流程、恢復 → 瓶頸轉移離開提示詞
- _關注點_：**代理基礎設施設計能力 = 下一個工程護城河**

**8. OpenAI × AWS $38B 協議**  
- 打破 Microsoft 專屬授權；多雲正式確立
- _關注點_：OpenAI API 採購路徑多元化，議價空間增加

### 🟡 觀望追蹤（3-6 個月）

**9. Gemma 4 + Qwen 3.6 開源崛起**  
- Gemma 4（Apache 2.0）：同規模 10 倍大模型效能；RAM 採納指數 >1（近期開源最高）
- Qwen 3.6：27B 密集模型領先開源排行榜（Artificial Analysis Index 46 分）
- _關注點_：自托管 LLM 方案成本效益持續提升

**10. OpenAI 自製手機計畫（2028）**  
- io 收購 $6.4B 為基礎；MediaTek、Qualcomm、Luxshare 合作
- 策略：AI 代理取代 App 生態
- _關注點_：2-3 年後裝置端代理可能改變 B2C 入口格局
