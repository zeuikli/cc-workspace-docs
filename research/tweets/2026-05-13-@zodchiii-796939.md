## 核心摘要

> Cloud Code 最佳實踐：從系統提示到自動化

### 主要內容
- Cloud Code 是純代理架構，通過指令、工具和迴圈工作
- 核心優勢：不用 RAG 向量索引，而是用 Grep/Find 等工具自動探索代碼庫
- 最佳實踐：使用 CLAUDE.md 文件進行跨會話狀態共享、權限管理自動化、上下文壓縮
- 進階用途：多會話並行、代理子任務分派、無頭自動化（CI/CD 集成）

### 關鍵洞察
Cloud Code 內部有完整的上下文管理（200K tokens）和自動化工具迴圈，開發者責任轉變為「PM 角色」：定義任務邊界、整理代碼庫知識而非逐行審查。本分享代表 Anthropic 團隊一年實戰的深度經驗總結。
